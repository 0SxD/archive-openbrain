## Creator Reference — PII Discipline

All notes, documentation, research files, agent outputs, and any content that may be shared or tracked in git must refer to the workspace owner as **"the creator"** or **"Sage"**.

Never use real names in:
- Tracked files (anything `git ls-files` returns)
- Files that may be staged for commit
- Research outputs
- Agent sync files
- Any content destined for GitHub or external sharing

Private/gitignored files (memory/, .claude/) may use real names since they stay local.
